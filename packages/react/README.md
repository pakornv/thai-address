# @thai-address/react

A React hook for managing Thai address selection with hierarchical data.

## Installation

```bash
deno add jsr:@thai-address/react
```

## Features

- **77 Provinces**: Complete data for all provinces, districts, subdistricts,
  and zip codes in Thailand
- **Hierarchical Selection**: Automatically filters available options based on
  parent selections
- **Zero Dependencies**: No runtime dependencies beyond React
- **Type Safety**: Full TypeScript support
